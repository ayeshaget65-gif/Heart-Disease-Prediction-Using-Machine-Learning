# Heart-Disease-Prediction-Using-Machine-Learning
This project demonstrates the use of machine learning algorithms to predict the likelihood of heart disease in patients based on health and lifestyle data. The dataset includes demographic, medical, and lifestyle variables such as age, gender, cholesterol, blood pressure, heart rate, smoking, alcohol intake, exercise, diabetes, obesity, 
Key Features of the Project:

Data Preprocessing & Cleaning: Handled missing values, encoded categorical variables for model training.
Model Training & Comparison: Implemented Logistic Regression, Support Vector Machine (SVM), and Random Forest Classifier.
Model Evaluation: Accuracy, confusion matrix, and classification reports to assess model performance.
Cross-Validation: Applied k-fold cross-validation to ensure robust and reliable model accuracy.
Feature Importance Analysis: Identified which factors contribute most to heart disease risk.
Visualizations: Confusion matrix and feature importance plots for better insights.
Optional Extension: Can be converted into a Streamlit interactive web app for real-time predictions.

Results:

Logistic Regression Accuracy: 86%
SVM Accuracy: 93%
Random Forest Accuracy: 100% (Note: may overfit due to small dataset)

Technologies Used: Python, Pandas, NumPy, Scikit-learn, Seaborn, Matplotlib

Takeaway: This project showcases end-to-end machine learning workflow, from data preprocessing to model evaluation and visualization, suitable for healthcare analytics applications.
